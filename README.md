# Insurance Risk, Claims & Customer Sentiment Intelligence Dashboard

## Overview

An end-to-end Business Intelligence project built using SQL Server, Power BI, Power Query, and Python.

The project analyzes insurance policies, claims, customer demographics, and customer feedback to generate actionable business insights.

## Technologies Used

* SQL Server
* Power BI Desktop
* Power BI Service
* Power Query
* Python
* DAX
* Row Level Security (RLS)

## Key Features

### Executive Dashboard

* Total Premium Amount Analysis
* Total Coverage Amount Analysis
* Total Claim Amount Analysis
* Active vs Inactive Policy Analysis
* Claim Status Distribution
* Premium Analysis by Policy Type

### Policy Details Explorer

* Drill-through Reporting
* Record-level Policy Analysis
* Customer and Claim Investigation

### Customer Sentiment Analysis

* Python-based Sentiment Processing
* Word Cloud Visualization
* Feedback Classification
* Sentiment Distribution Analysis

## Advanced Features

* SQL Server Integration
* Drill-through Navigation
* Row Level Security (RLS)
* Power BI Service Publishing
* Dashboard Creation
* Python Script Integration

## Dataset

Insurance dataset based on a Muthoot Finance business scenario containing policy, customer, claim, and feedback information.

## Project Outcomes

* Built an interactive insurance analytics solution.
* Implemented secure role-based access using RLS.
* Converted customer feedback into sentiment insights.
* Enabled detailed policy investigation through drill-through reporting.
